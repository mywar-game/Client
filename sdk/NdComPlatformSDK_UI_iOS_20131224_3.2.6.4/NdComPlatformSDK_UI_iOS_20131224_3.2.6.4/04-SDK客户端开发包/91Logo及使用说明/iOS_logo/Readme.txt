
为了标识接入了“91移动开放平台”，开发者可以在应用图标上加入平台logo标识。

四个子文件夹，分别是四个角落的logo类型，所以您只要选择一组就可以了，原则上以不影响原图标功能、美观为前提，请选择对应的logo模板。
不要修改91logo的大小。开发者必须严格按照下面尺寸使用logo。

应用图标类

普通iphone，ipod touch 使用【57 x 57】
高清iphone，ipod touch 使用【114 x 114】
普通ipad 	       使用【72 x 72】
高清ipad               使用【144 x 144】
上架时候 	       使用【512 x 512】

其他的用于Spotlight和设置
